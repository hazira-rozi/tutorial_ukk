Tentang Aplikasi

Aplikasi ini dibangun menggunakan Laravel 10/11 dan Bootstrap SB Admin 2. Fitur utamanya mencakup manajemen inventaris motor, transaksi peminjaman, pengembalian otomatis dengan denda, dan laporan.

Laravel MySQL Bootstrap

1. Persiapan Tools

Sebelum memulai, pastikan tools berikut sudah terinstall.

01

Laragon (Web Server)

Laragon digunakan sebagai paket lengkap (PHP, MySQL, Apache/Nginx).

  1. Download Laragon Full di laragon.org.
  2. Install dan jalankan Laragon.
  3. Klik tombol Start All untuk menjalankan Service Apache dan MySQL.
  4. Pastikan PHP versi >= 8.1 (Cek di Menu > PHP > Version).
02

DBeaver (Database Manager)

Digunakan untuk mengelola database MySQL secara visual.

  1. Download DBeaver Community di dbeaver.io.
  2. Install dan buka aplikasi.
  3. Buat koneksi baru: Pilih MySQL.
  4. Server Host: localhost, Port: 3306.
  5. Username: root, Password: (kosongkan jika default Laragon).
03

Visual Studio Code

Text Editor utama.

Install Extension wajib:

  • PHP Intelephense
  • Laravel Blade Snippets
  • Laravel Extra Intellisense

2. Instalasi Project

Langkah-langkah mengambil kode dari GitHub dan menyiapkan dependensi.

Langkah 1: Clone Repository

Buka Terminal (Git Bash atau CMD) di folder C:\laragon\www.

Terminal 
cd C:\laragon\www
git clone https://github.com/hazira-rozi/rental_motor.git
cd rental_motor

Langkah 2: Install Dependensi PHP

Jalankan perintah composer untuk mengunduh library Laravel.

Terminal 
composer install

Langkah 3: Konfigurasi Environment

Duplikat file contoh konfigurasi.

Terminal 
cp .env.example .env
php artisan key:generate

3. Konfigurasi Database

Menghubungkan aplikasi dengan MySQL via DBeaver.

Buat Database Baru

Buka DBeaver, klik kanan pada koneksi Localhost -> Create New Database.

  • Database Name: db_rental_motor
  • Charset: utf8mb4

Edit File .env

Buka file .env di VS Code, cari bagian DB dan sesuaikan:

.env 
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=db_rental_motor
DB_USERNAME=root
DB_PASSWORD=

Migrasi & Seeding

Membuat tabel dan data dummy awal (Admin).

Terminal 
php artisan migrate:fresh --seed

4. Perbaikan Kode (Penting)

Berdasarkan analisa, ada beberapa file yang perlu dibersihkan/diperbaiki.

1. Hapus File Sampah (question.js)

File public/js/questions/question.js berisi logika ujian online yang tidak relevan. Silakan hapus file tersebut agar proyek bersih.

2. Perbaikan RentalController.php (Approve)

Tambahkan validasi stok motor sebelum menyetujui peminjaman untuk keamanan ganda.

RentalController.php
// Method approve() yang diperbaiki
public function approve($id) {
    $rental = Rental::with('motor')->findOrFail($id);

    if ($rental->status_transaksi == 'menunggu') {
        // Cek apakah motor benar-benar tersedia/disewa statusnya
        if($rental->motor->status !== 'disewa') {
             return back()->with('error', 'Status motor tidak valid.');
        }

        $rental->update(['status_transaksi' => 'proses']);
        return back()->with('success', 'Peminjaman disetujui.');
    }
    return back()->with('error', 'Aksi tidak valid.');
}

Terminal
